Kolla ♥ AoC

At Kolla we love Advent of Code! We also love helping developers write integrations faster and more securely! So we built Konnect, an Integrations Platform that helps you integrate your software with your customers SaaS products!

Read more about Konnect on our docs pages: Konnect Docs

Join Our Raffle

After you set a record time on your next AoC puzzle, take a few minutes and build a demo slack integration with Konnect! If you complete the steps below you will get entered into our raffle for a chance to win a DJI Mini Drone 2!

Complete these steps to automatically enter the raffle

1. Create your Kolla Account
Set up your Kolla account: https://app.getkolla.com/signup
2. Generate your Kolla API Key

Generate a new API key in your Kolla account. You can find this under Settings > API Keys.

3. Install Konnect

Follow the docs to install Konnect: https://docs.getkolla.com/kolla/getting-started/installing-konnect

4. Create a Slack integration

Use the Slack connector that is pre-setup with your Kolla account to create a new Slack integration. Once you have a linked account setup you are automatically entered into the raffle!
